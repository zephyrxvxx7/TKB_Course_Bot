TKB幹課
===========================


## 環境需求
* python 3.6.3 up
* selenium
* chromedriver
	* https://sites.google.com/a/chromium.org/chromedriver/downloads
* chrome正式發行版本

## 如何使用？
### 在 info.py 填好以下資料
#### 輸入帳號密碼
	username = ''
    password = ''
#### 輸入想要幹的選課名稱
	classID = '資料結構'
#### 輸入日期
	year = '2018'
	month = '07'
	day = '24'
#### 輸入地點
	branch = '東海'
#### 搶課Flag設為True
	grabbed = True
### 在完美的時間執行 TKB_Scraper.py 即可成功（吧
